Instructions

Overview

This Admin Dashboard will help the Mindful Menu team build and manage menus and users. Below are instructions for how to add meals for upcoming weeks.


Managing Users (Beta Testers)

While doing beta testing of the app, the process to enable a user's account is manual, by the Mindful Menu team. A team member will need to verify each user through this dashboard. The process is as follows:

  1. User will go to the App Store and install the app.
  2. They'll open the app, click "Sign Up", enter their info, and create an account.
  3. On the next page, if they already have an active subscription, they can click "I'll do it later". If not, they can use the links to enter payment info, and subscribe.
  4. The user then will see a screen telling them their account is pending, and a team member will send them an email within 2 days when account is active.
  5. As an admin, you'll login to this dashboard every 1-2 days, and look over the users whose account is "disabled".
  6. For each, check that they have an active subscription (Moon Clerk) and that they have verified their email address, and if yes to both, then click "Edit" for that user and check the box for "Enabled".
  7. The next time that user opens their app, they should see the regular screens, and can start using it.

Adding a new week's menu

To add a new week with 7 meals, you first need to create the meals, and then will create the new menu that references those meals.

To create new meals, click on "Meals" in the navigation bar above, and click "Add Meal" button in the upper right. Do this for all the new meals.

After the meals are created, click on "Weekly Menus" in the navigation bar above, and click "Add Weekly Menu" button in upper right. Enter the dates for the new menu, and then lookup each new meal you created, by name. When you start typing in the "Meals" section, a list of matching meal names will show, and you can select the one you want.

Once all the meals are add, and the new menu has been created, you are done, and that menu is ready to be published for users to see.

NOTE: If you are adding your own image of a meal, we're hosting those files under the Google Drive of the ourmindfulmenu@gmail.com account. Watch this video to learn how to get link to image. That base URL before the ID is: https://drive.google.com/uc?id=


Expectations

Publication of new menus

All weekly menus need to be completed BEFORE 4pm Thursday, of the week prior. The users will start seeing the next week's menu at 4pm on Thursday, so it needs to be finalized by the MM Team prior to that. There is no limitation on how early a weekly menu can be created in the system (i.e. you can create menus months in advance, and they just won't show up until the scheduled week.).
